Character Sheet

Esse projeto foi criado para facilitar o gerenciamento de personagem, pois, usar papel é exaustivo de ficar apagando a vida e mudando alguma coisa. Então, porque não criar um aplicativo para facilitar, e também porque não colocar uma página que mostrar os dados rolando e botar ele na tv, em vai decidir que o personagem vai ter azar ou sorte.

Nesse projeto, eu utilizei a linguagem Ruby e os framework Ruby on Rails e Boostrap e além disso, usei recursos como Action cable para usar o websocket para rolar os dados e também o GM ver as vidas do seus jogadores participantes e Postgresql como banco de dados.

As gems e biblioteca utilizado foram:
Clearance para autenticação com email e senha;
Friendly id para deixar a url mais bonita;
Redis para rodar action cable;
Dice-Box3d para ter as animações dos dados.

Apesar de eu considerar que já está concluído, ainda falta muita coisas, como magias, sistema de ataque etc.

Infelizmente, não está no ar, pois, o heroku, a qual eu usava para hospedar, terminou com serviço gratuito. Então, unica forma que você pode ver isso rodando na sua maquina. Eu coloquei um passo a passo no github, clique aqui.
